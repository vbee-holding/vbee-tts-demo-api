# vbee-tts-demo-api

## Installation

```
npm install
```

## Running

```
npm start
```

## Api docs

| Method | Uri                     | Action                 |
| ------ | ----------------------- | ---------------------- |
| GET    | /voices                 | Get list of voice      |
| GET    | /synthesis-techs        | Get list of tech       |
| GET    | /tts?voice=&input_text= | Convert text to speech |
